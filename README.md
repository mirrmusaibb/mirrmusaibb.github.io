# Mir Musaib — Digital Observatory

A handcrafted personal website for **MIR MUSAIB**: a calm, cinematic digital homepage inspired by modern observatories, research labs, knowledge networks, and premium product design.

## Stack

- HTML
- CSS
- JavaScript
- Three.js for the neural constellation background
- GSAP + ScrollTrigger for elegant reveal motion

## Structure

```text
index.html
css/
  style.css
js/
  main.js
  background.js
  animations.js
assets/
```

## Experience

The site includes:

- Full-screen cinematic hero
- Three.js neural constellation background
- Premium glassmorphism dashboard cards
- Future-facing exploration section
- Minimal about section
- Responsive layouts for desktop, tablet, and mobile
- Accessibility basics including semantic HTML, skip link, keyboard focus, contrast, and reduced-motion support

## Running locally

Open `index.html` directly in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
